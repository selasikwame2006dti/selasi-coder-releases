# Selasi Coder — Releases

Downloads for the Selasi Coder desktop app. Grab the latest installer from the **[Releases](https://github.com/selasikwame2006dti/selasi-coder-releases/releases/latest)** page, or from [selasi.dev](https://selasi.dev).

**Current stable: v1.1.0.** If you're on anything older than v1.1.0, please update — earlier versions had known reliability issues that are now fixed.

## ⚠️ Windows SmartScreen warning

The installer isn't signed with a paid Microsoft certificate yet, so Windows will show **"Windows protected your PC"** when you run it. This is expected, not a sign anything's wrong:

1. Click **"More info"**
2. Click **"Run anyway"**

A full walkthrough (with screenshots) is available at [selasi.dev/install](https://selasi.dev/install).

## What is this?

Selasi Coder is a local AI coding agent — point it at a folder on your PC and it reads, writes, and runs commands directly in your project. It also supports autonomous overnight runs, browser automation, and (opt-in) PC control for tasks beyond code.

It talks to the hosted API at `https://selasai.alwaysdata.net/api/coder` — your account, credits, and login are shared with the web app at [selasi.dev](https://selasi.dev).

## Auto-updates

The app checks for new versions automatically and will prompt you to install when one's available — you're always just one click from the latest release.
